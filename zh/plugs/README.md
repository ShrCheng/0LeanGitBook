# 第四章&nbsp;&nbsp;&nbsp;GitBook 个性化及插件
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;个性化主要是对书籍的一些设置,比如设置书籍的标题,作者等等.插件主要的针对于 GitBook 使用的插件(针对于该书籍),以及对插件的配置.
##### 个性化示例
---
```
"author": "ShiCheng",
"output": null,
"generator": "site"
```
##### 插件示例
---
```
"plugins": [
    "mermaid-2"
    ],
"pluginsConfig": {
    "mermaid-2": {
        "theme": "forest"
    }
}
```
